<p id="description">KDS Fitness is a personal fitness and physique transformation command center. It combines workout logging nutrition tracking body metrics progress photos and daily habit tracking into a single dashboard — then gamifies the whole journey with an XP/leveling system unlockable achievements and Solo Leveling-inspired hunter ranks (E through S-Rank up to "Shadow Monarch"). Set a dream physique goal break it into roadmap phases and track your progress toward it day by day.</p>

<h2>🧐 Features</h2>

Here're some of the project's best features:

*   🏠 Dashboard — at-a-glance view of current body metrics dream physique goal progress and power level (XP/level)
*   🗺️ Transformation Roadmap — break your goal into phases (Foundation Fat Loss Lean Bulk Final Cut) with editable progress and notes
*   🏋️ Workout Logging — log exercises sets reps and weight per session
*   🍽️ Nutrition Tracking— log meals with calories/protein/carbs/fats against daily targets
*   💧 Sleep & Water Tracking — daily hydration and sleep logging with 7-day trend charts
*   📸 Progress Photos — visual timeline of your transformation
*   📊 Body Metrics — track weight body fat % waist and shoulder measurements over time with charts
*   ✅ Habit Tracker — streak-based tracking for custom daily habits
*   🎮 Gamification ("The System") — earn XP for logging activity level up unlock achievements and climb Solo Leveling-style hunter ranks (E → D → C → B → A → S → Shadow Monarch)
*   🌓 Dark/Light theme
*   💾 All data persists locally (no backend/account required)

<h2>🛠️ Installation Steps:</h2>

<p>1. Clone the repository</p>

```
git clone https://github.com/your-username/your-repo.git
```

<p>2. Install dependencies</p>

```
bun install
```

<p>3. Start the dev server</p>

```
bun run dev
```

<p>4. Open in browser</p>

```
http://localhost:3000
```

  
  
<h2>💻 Built with</h2>

Technologies used in the project:

*   Framework: \[TanStack Start\](https://tanstack.com/start) (React 19 file-based routing via TanStack Router)
*   State: Zustand (with localStorage persistence)
*   Styling: Tailwind CSS v4 Radix UI primitives shadcn-style component kit
*   Charts:Recharts
*   Animation: Framer Motion
*   Deployment: Cloudflare (via \`@cloudflare/vite-plugin\` + Wrangler)
*   Built with \[Lovable\](https://lovable.dev)
